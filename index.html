FEATURED PROJECT

Lost & Found Portal

A secure full-stack platform designed to simplify reporting, searching, and recovering lost belongings through authentication, intelligent search, image uploads, and real-time communication.

Featured Case Study

Helping students reconnect with their belongings.

Built as a MERN Stack application, the Lost & Found Portal provides a centralized platform where users can report lost or found items, communicate securely, search community listings, and manage everything from a personalized dashboard.

01

The Problem

Students frequently lose important belongings across campus. Most recovery attempts rely on word of mouth, WhatsApp groups, or social media posts, making the process slow, unorganized, and often unsuccessful.

02

The Solution

I built a secure MERN Stack application where users can report lost or found items, upload images, search community listings, communicate through an integrated messaging system, and manage all activities from a personalized dashboard.

03

Engineering Decisions

  • JWT Authentication for secure access
  • MongoDB for flexible item storage
  • REST APIs using Express.js
  • Responsive UI for mobile and desktop
  • Reusable React component architecture
04
